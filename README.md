# ZadAI

### Project Overview

the project consists of the development a IA assistant totally local, able to voice conversation in real time, understand natural language, access memory persistence storage in a vault of Obsidian and execute task in operational system.

all architecture executed in archLinux with a GPU RTX 3050 6GB and 16GB RAM.

the main object its crate a system like a JARVIS, independent cloud services.

### Project Objects

* Natural voice conversation
* Fully local/offline execution
* Obsidian-based persistent memory
* Contextual organization of projects and notes
* Semantic search in Markdown files
* Linux system automation
* Modular and expandable architecture
* Low VRAM consumption
* Compatibility with mid-range hardware
